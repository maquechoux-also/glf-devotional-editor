Events

One card per event — pick a color theme, set the date, and write the details. Events are automatically ordered soonest-first when the email is built.

In any text box below: wrap words in **double asterisks** for bold navy emphasis, or *single asterisks* for italics. To link to a webpage, highlight the words and click the 🔗 Add Link button that pops up (or type [words](https://page)). Separate paragraphs with a blank line — the first paragraph is slightly larger, the rest are compact sub-lines.
